About Liferea

This documentation applies for version 0.9.6 of Liferea. This documentation might not yet be perfect. If you want to help updating the documentation please contact us on the mailing list!

Liferea is an abbreviation for Linux Feed Reader. It is a news aggregator for online news feeds. It supports a number of different feed formats including RSS/RDF, CDF, and Atom 1.0.

If you are interested in the recent Liferea versions, want to submit a bug or request a feature you may want to visit the project page at SourceForge! You are also invited to join the liferea-devel mailing list. Details on the list are located at The SourceForge list info page.
